PAndQ.TruthTable — Type
TruthTable(ps)

Construct a truth table for the given Propositions and LogicalOperators.

The header is a vector containing vectors of logically equivalent propositions. The sub_header corresponds to the header, but contains each proposition's UnionAll type. The body is a matrix where the rows contain interpretations of each proposition in the given column.

See also tautology and contradiction.

Examples

julia> TruthTable([Tree(⊤)])
┌──────┐
│ ⊤    │
│ Tree │
├──────┤
│ ⊤    │
└──────┘

julia> @atomize TruthTable([¬p])
┌──────────┬─────────┐
│ p        │ ¬p      │
│ Variable │ Literal │
├──────────┼─────────┤
│ ⊤        │ ⊥       │
│ ⊥        │ ⊤       │
└──────────┴─────────┘

julia> @atomize TruthTable([p ∧ ¬p, p ⊻ q, ¬(p ∧ q) ∧ (p ∨ q)])
┌────────┬──────────┬──────────┬───────────────────────────┐
│ p ∧ ¬p │ p        │ q        │ p ⊻ q, ¬(p ∧ q) ∧ (p ∨ q) │
│ Tree   │ Variable │ Variable │ Tree, Tree                │
├────────┼──────────┼──────────┼───────────────────────────┤
│ ⊥      │ ⊤        │ ⊤        │ ⊥                         │
│ ⊥      │ ⊥        │ ⊤        │ ⊤                         │
├────────┼──────────┼──────────┼───────────────────────────┤
│ ⊥      │ ⊤        │ ⊥        │ ⊤                         │
│ ⊥      │ ⊥        │ ⊥        │ ⊥                         │
└────────┴──────────┴──────────┴───────────────────────────┘

Base.show — Function
show(::IO, ::MIME"text/plain", ::Proposition)

Represent the given Proposition as a propositional formula.

Examples

julia> @atomize show(stdout, MIME"text/plain"(), p ⊻ q)
p ⊻ q

julia> @atomize show(stdout, MIME"text/plain"(), Normal(p ⊻ q))
(p ∨ q) ∧ (¬p ∨ ¬q)

show(::IO, ::MIME"text/plain", ::TruthTable)

julia> @atomize show(stdout, MIME"text/plain"(), TruthTable([p ∧ q]))

show(::IO, ::Proposition)

Represent the given Proposition expanded as valid Julia code.

Examples

julia> @atomize s = sprint(show, p ∧ q)
"Tree(and, Tree(identity, Variable(:p)), Tree(identity, Variable(:q)))"

julia> @eval $(Meta.parse(s))
p ∧ q

AbstractTrees.print_tree — Function
print_tree(::IO = stdout, ::Proposition; kwargs...)

Prints a tree diagram of the given Proposition.

Note

Instances of Compound{typeof(identity)} are represented as I.

See also AbstractTrees.print_tree.

julia> @atomize print_tree(p ∧ ¬q ⊻ s)
⊻
├─ ∧
│  ├─ I
│  │  └─ p
│  └─ ¬
│     └─ q
└─ I
   └─ s

julia> @atomize print_tree(Normal(p ∧ ¬q ⊻ s))
∧
├─ ∨
│  ├─ I
│  │  └─ p
│  └─ I
│     └─ s
├─ ∨
│  ├─ ¬
│  │  └─ q
│  └─ I
│     └─ s
└─ ∨
   ├─ ¬
   │  └─ p
   ├─ I
   │  └─ q
   └─ ¬
      └─ s
